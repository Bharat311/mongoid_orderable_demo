Mongoid Orderable Demo App
==========================

A sample application demonstrating the use of the mongoid orderable gem using Rails 3.2.18

##Requirements

* Rails - 3.2.18
* Ruby  - 1.9.3 (or more)
* Mongoid - 3.1.6 (or more)

## Setup

* Clone the repository
* Run `bundle install`
* Run `rake db:seed`
* Now visit the app at 'http://localhost:3000'
